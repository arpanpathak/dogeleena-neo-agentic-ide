# Vim Motions Quick Reference

## Movement

| Key | Motion |
|-----|--------|
| `h/j/k/l` | Left/Down/Up/Right |
| `w/b` | Word forward/back |
| `0/$` | Start/End of line |
| `gg/G` | Start/End of file |
| `Ctrl+d/u` | Half page down/up |

## Editing

| Key | Action |
|-----|--------|
| `i` | Insert |
| `a` | Append |
| `dd` | Delete line |
| `yy` | Copy line |
| `p/P` | Paste after/before |
| `u/Ctrl+r` | Undo/Redo |
| `.` | Repeat change |

## Visual

| Key | Action |
|-----|--------|
| `v` | Character select |
| `V` | Line select |
| `Ctrl+v` | Block select |
| `y` | Copy |
| `d` | Delete |

## Search

| Key | Action |
|-----|--------|
| `/pattern` | Search forward |
| `?pattern` | Search backward |
| `n/N` | Next/prev match |
| `*` | Search word under cursor |
| `:%s/old/new/g` | Replace all |
